## Card Images

Card graphics live in `cards/`.

- `card-face.png`
- `card-back.png`

The card ranks and suits are still rendered by code on top of the face image, so you can replace the card texture without breaking gameplay.

## Dealer Image

Dealer art lives in `dealers/`.

- `dealer-neutral.png`

Replacing this file changes the dealer shown on every standard blackjack table.

## Free Bet Token

Free Bet coin art lives in `tokens/`.

- `free-bet-coin.png`

This is the golden coin used for Free Bet / Pot of Gold UI. Replace it with the same filename to customize it.
