# Basic Card Counting Guide

A common beginner system is **Hi-Lo Count**.

### Card Values:

| Card | Count Value |
|------|-------------|
| 2,3,4,5,6 | +1 |
| 7,8,9 | 0 |
| 10,J,Q,K,A | -1 |

### How It Works:

- Low cards increase the count
- High cards decrease the count
- Higher positive counts may favor the player
- Negative counts may favor the house

### Example:

If these cards are seen:

5, K, 3, A, 9

Count becomes:

+1  
-1  
+1  
-1  
0

Running Count = **0**
